# Neutron Interaction with Silicon using FLUKA

## Overview
This project presents a simulation study of neutron interaction with silicon material using the FLUKA Monte Carlo method. The analysis focuses on particle fluence and energy deposition resulting from neutron irradiation.

## Objectives
- To analyze neutron interaction with silicon
- To evaluate particle fluence distribution
- To study energy deposition within the material

## Data Description
The dataset includes:
- Particle fluence (proton, electron, neutron, positron, photon)
- Energy deposition values
- Distribution along X and Z axes

All processed data is available in the Excel file.

## Methodology
- Simulation performed using FLUKA
- Monte Carlo-based particle transport
- Data processed using Microsoft Excel

## Key Insights
- Neutron shows higher fluence due to strong nuclear interaction
- Positron has lower fluence due to annihilation process
- Energy deposition occurs through complex particle interactions
